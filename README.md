# HIO-SNOMEDCT-AML
Semantic interoperability study between HIO and SNOMED CT in the hearing loss domain.

## Ontologies

### Hearing Impairment Ontology (HIO)

Available at:
https://bioportal.bioontology.org/ontologies/HIO

### SNOMED CT

Available at:
https://www.snomed.org

Version used:
SNOMED CT International Release 2024-09

## Alignment Tool

AgreementMakerLight (AML)

https://github.com/AgreementMakerLight/AML-Project

## Experimental Configuration

- Similarity threshold: 0.5
- HIO concepts analyzed: 495
- SNOMED CT concepts analyzed: 797
- Valid matches identified: 39

## Reproducibility

Researchers may obtain the ontologies from their official sources and reproduce the experiments using the configuration described above.
